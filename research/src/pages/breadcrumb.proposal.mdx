## Overview

Breadcrumbs are navigation land marks used to show a user's location within the site or app.

## Use Cases

A `<breadcrumb />` trail consists of a list of links to the parent pages of the current page in hierarchical order. It helps users find their place within a website or web application. Breadcrumbs are often placed horizontally before a page's main content.

## &lt;breadcrumb&gt; Properties <a href="#breadcrumb-properties" id="breadcrumb-properties"></a>

| Attribute Name | Type             | Default Value | Description                                                                      |     |
| -------------- | ---------------- | ------------- | -------------------------------------------------------------------------------- | --- |
| `items`        | `HTMLCollection` |               | A `HTMLCollection` of items to represent the current path within the application |     |
| `divider`      | `custom`         | `/`           | Represnets the custom item to be used as divider between the items               |     |

#### Structure <a class="link" href="#structure" id="structure"></a>

- `<breadcrumb>` - The root element that contains the list of items and dividers **[required]**
- `<breadcrumb-item>` - The element representing a place in the path to the current position on the application **[required]**
- `<breadcrumb-divider>` - The divider between the `<breadcrumb-item />`

#### Accessibility <a class="link" href="#accessibility" id="accessibility"></a>

It should be wrapped by a navigation landmark `nav` conatining `aria-label` or `aria-labelledby`, the `<breadcrumb-item />` representing the current page should contain `aria-current="page"` ( if the current page isn't a link the attribut is optional )
